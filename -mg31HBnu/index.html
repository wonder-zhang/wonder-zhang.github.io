    • 自建网易云音乐破版权服务器

    首先你的准备一台服务器不用太高配置，512MB内存1核即可。
    服务器安装好宝塔面板，然后在宝塔面板的软件管理页面安装PM2管理器

    然后在终端执行命令：

    git clone https://github.com/nondanee/UnblockNeteaseMusic.git

    他的默认端口是8080，自己自行更改即可，我没有改，直接弄的8080端口
    修改好后点击安全，放行端口，这就像开门迎客，你门都不开，那怎么让客人来

    回到PM2管理器中，设置，根据下图添加即可

    记得要给项目添加个名称在点击添加！！
    然后在客户端设置相应的代理，然后确定


    好啦！这样就大功告成了，如果还是提示错误，那就要修改host文件了。将前面参数修改为服务器IP

    你的服务器ip music.163.com
你的服务器ip interface.music.163.com

    很多人也许会遇到pm2中的管理进程莫名其妙会自动关闭的现象，其实，这个正常，你的端口冲突了，兄弟!!!也就是说，这个端口正在被占用，你该换一个别的端口去。阿里云服务器的用户，记得打开你所需防火墙的端口，要不然联通能联通，就是没办法加载！！！
