    • aria2安装实录，利用termux实现高速下载

    今天用termux尝试安装aria2实现一些文件的高速下载。
    好吧现在开始为了方便我直接用的电脑ssh连接

    ssh连接电脑

    1.安装openssh
    pkg install openssh
    2.开始ssh服务
    sshd

    3.电脑连接，在这里我用的xshell，termux用的是8022端口

    现在开始正课

    1.升级软件包pkg upgrade
    2.然后安装aria2的文件
    pkg install aria2
    3.使用la命令找到aria2的配置文件，因为是默认不显示的

    4.进入.config里面的aria2目录,并对配置文件进行编辑,省略掉vim aria2 这一行命令，我手快敲错了

    5.具体配置文件是这样的（vim编辑器不会用的可以自行百度）

    5.5配置好之后键入如下命令行，是图片中的那样就是启动成功了。
    aria2c --enable-rpc --rpc-listen-all

    接下载再次新打开一个窗口

    6.git一下web的下载配置

    git clone https://github.com/ziahamza/webui-aria2.git

    7.cd到对应目录cd webui-aria2
    8.启动web界面node node-server.js

    9.在浏览器输入你的ip:8080 并且打开设置>连接设置进行设置。。。。（这话有点绕口啊~）

    出现如下字样表示连接成功

    10.进行下载
